After I decided my brain wouldn't retain anything if I moved onto the next section I logged into my even more neglected codewars account and solved the Isograms challenge.

The Isograms challenge requires that you take a string and return true if each letter in the string is present only one time. If the letter is present more than once (consecutive or not), then the function should return false. Immediately I knew I would need a Regular Expression. I also knew that my regex skills were rusty. I couldn't think of how to write a regex that would test for a single occurance of a letter. So what I decided to do was to first split the string into an array, then using a forEach loop, use each letter in a regular expression, and using .match() return false if the array returned from .match() was longer than 1.

The solution worked, but I wished I had remembered how to use regular expressions to solve the challenge in fewer steps.
